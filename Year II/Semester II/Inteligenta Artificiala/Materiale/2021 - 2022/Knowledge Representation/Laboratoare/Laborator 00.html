Reprezentare grafuri¶

Grafuri neorientate¶

Matrice de adiacenta¶

Contine pentru fiecare pereche de noduri din graf un indicator binar care indica daca intre cele doua noduri exista sau nu o muchie.

Daca intre nodul i si nodul j exista o muchie in graf, atunci m[i][j] = 1.

Astfel, fiecare rand i (sau coloana i) al matricei gasim pe pozitiile j unde este valoarea 1, nodurile j care au o muchie cu i.

Pentru grafurile neorientate (ca cel din exemplu), matricea de adiacenta are proprietatea ... .

Grafuri cu muchii ponderate¶

Atunci cand unei actiuni de tranzitie intre stari ii este asociat un cost, atunci muchiile / arcele vor avea asociata o pondere.

In acest caz, in matricea de adiacenta putem avea numere reale care reprezinta costul in loc de valori binare 0/1.

      [0, 1.5, 0.1, 0.34],
      [1.5, 0, 0, 3],
      [0.1, 9, 0, 0],
      [0.34, 3, 0, 0]
    ]

Parcurgerea in latime: BFS (breadth-first search)¶

Pasii algoritmului¶

Initializari. Trebuie sa stabilim un nod de start si unul sau mai multe noduri scop. In plus consideram o lista de noduri (sau drumuri, adica liste de noduri) neexpandate inca, organizata sub forma de coada (o vom nota cu c sau frontiera). Punem nodul de start in coada.

Repetam urmatorii pasi atata timp cat coada nu e vida, sau nu ne-am oprit din cautat drumuri (am calculat deja suficiente drumuri-solutie):

  1. Extragem primul nod din coada. In cazul in care memoram chiar drumuri, extragem drumul.
  2. Verificam daca e nod scop. Daca da, ne oprim. (in cazul in care ne intereseaza drumul in sine, afisam drumul)
  3. Expandam nodul extras (sau ultimul nod din drumul extras) obtinand nodurile lui vecine
  4. Daca nodurile nu au mai fost vizitate, le adaugam in coada c. Pentru a marca nodurile vizitate, verificam pentru fiecare nod din drumul curent (de la radacina pana la nodul tocmai expandat) daca mai apare in drum, caz in care nu il adaugam in coada).

Parcurgerea in adancime: DFS (depth-first search)¶

Pasii algoritmului¶

Initializari. Trebuie sa stabilim un nod de start si unul sau mai multe noduri scop. Consideram o stiva S initial vida. Punem nodul de start in stiva S.

Repetam urmatorii pasi atata timp cat stiva nu e vida, sau nu ne-am oprit din cautat drumuri (am calculat deja suficiente drumuri-solutie):

  1. Pentru primul nod din stiva verificam daca e nod-scop, caz in care afisam drumul. Putem continua apoi algoritmul (in cazul in care avem mai multe noduri scop, putem avea un drum care se termina cu un nod-scop dar are si alte cateva astfel de noduri in componenta sa.)

  2. Pentru primul nod din stiva cautam un succesor care nu a mai fost evaluat pana acum din multimea de succesori ai nodului din varful stivei, si care nu se afla in stiva curenta (e nevoie si de a doua precizare fiindca nodul succesor daca se afla in stiva curenta, a fost evaluat anterior, dar cu alt parinte). Il punem in stiva, succesorul devenind astfel noul varf.

  3. Daca nu avem succesori pentru primul nod din stiva, il stergem din stiva (spunem ca se realizeaza o intoarcere).

def matrix_to_dict(ma):
  d = dict()
  n = len(ma[0])
  for x in range(0, n):
    d[x + 1] = []
  for index in range(0, n):
    for ind in range(0, n):
      if ma[index][ind]:
        d[index + 1].append(ind + 1) 
  return d

def BFS(mo, nod_start, nod_scop):
  frontiera = [] # coada
  parcurse = set()

  frontiera.append(nod_start - 1)
  parcurse.add(nod_start - 1)

  while len(frontiera) > 0:
    x = frontiera.pop(0)
    print(x, end = " ")
    if x == nod_scop - 1:
      break
    else:
      for i in range(0, len(mo[x])):
        if mo[x][i] > 0 and (i not in parcurse):
          frontiera.append(i)
          parcurse.add(i)
  return

def depth_limited_DFS(mo, nod_start, nod_scop, niv_max):
  frontiera = [] # stiva
  parcurse = set()  
  n = len(mo[0])
  nivel = [0  for i in range(n + 1)]
  frontiera.append(nod_start - 1)
  parcurse.add(nod_start - 1)
  nivel[nod_start] = 1
  while len(frontiera) > 0:
    n = len(frontiera) - 1
    x = frontiera.pop(n)
    print(x, end = " ")
    if x == nod_scop - 1 or nivel[x] == niv_max:
      break
    else:
       for i in range(0, len(mo[x])):
        if mo[x][i] > 0 and (i not in parcurse):
          frontiera.append(i)
          nivel[i] = nivel[x] + 1
          parcurse.add(i)
